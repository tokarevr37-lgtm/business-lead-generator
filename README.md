# Business Lead Generator

A Python automation tool for collecting and organizing business contact data for lead generation, market research, and outreach workflows.

## What this project does

This project demonstrates a practical lead generation workflow that organizes structured business contact data into clean CSV and Excel deliverables.

Collected business data includes:

- Company name
- Website
- Business email
- Phone number
- City
- Industry category

## Features

- Business lead data organization
- CSV export
- Excel export with formatting
- Structured contact list output
- Reusable workflow for lead generation tasks
- Clean client-ready deliverables

## Tech Stack

- Python
- OpenPyXL
- CSV
